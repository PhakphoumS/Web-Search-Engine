<h3>CAVE2/Unity Simulator</h3>
<div id='entry_image_large'><img style='left:0;' src='/resizedImages/1280x720-cave2unitysimulator.png' alt='' title='CAVE2/Unity Simulator' width='640' height='327'></div>
<span class='smallText'>CAVE2/Unity Simulator being presented by EVL PhD candidate Arthur Nishimoto. - L. Long, EVL/UIC</span><br><br>
<div class='entry_content'>
	<strong>Researchers:</strong> <a href='/entry.php?id=270'>Arthur Nishimoto</a><br><br>
	<strong>URL:</strong> <a href='https://github.com/uic-evl/omicron-unity' target='_blank'>https://github.com/uic-evl/omicron-unity</a><br><br>
<!-- Begin Entry Description -->
Developed by EVL PhD Candidate Arthur Nishimoto the CAVE2/Unity simulator is a Unity3D package for testing and development of Unity3D applications for the CAVE2&trade; Hybrid-Reality Environment.  Using only standard Unity features (no platform-dependent libraries or Unity Professional license required), this package acts as a wrapper around the CAVE2 API to simulate the CAVE2 viewport and 3D spatial wand interactions on a desktop/laptop development environment.<br><br>
The associated <a href="https://github.com/arthurnishimoto/module-omicron">module-omicron package</a> also serves as the Unity/C# interface of accessing the omicron input abstraction library.<br><br>
<!-- End Entry Description -->

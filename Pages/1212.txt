<h3>OmegaDesk: An Instrument for Interactive Visual Data Exploration and Collaboration</h3>
<div id='entry_image_large'><img style='left:0;' src='/resizedImages/1280x720-_ljl4307.jpg' alt='' title='OmegaDesk: An Instrument for Interactive Visual Data Exploration and Collaboration' width='640' height='360'></div>
<span class='smallText'>OmegaDesk enables users to view stereoscopic 3D volumes while browsing - L. Long, EVL</span><br><br>
<div class='entry_content'>
	<strong>Researchers:</strong> <a href='/entry.php?id=10'>Andrew Johnson</a>, <a href='/entry.php?id=21'>Jason Leigh</a>, <a href='/entry.php?id=281'>Victor Mateevitsi</a>, <a href='/entry.php?id=250'>Yiwen Sun</a>, Alessandro Febretti<br><br>
	<strong>Funding:</strong> NSF Award CNS-0821121<br><br>
<!-- Begin Entry Description -->
OmegaDesk provides researchers with a powerful, easy-to-use, information-rich, cyberinfrastructure instrument in support of scientific discovery. The OmegaDesk unifies ultra-high-resolution computer-enhanced collaboration workspaces and stereoscopic virtual environments with multi-touch-sensitive surfaces so that users can intuitively point, write, touch and manipulate the information displayed, and communicate and share this information with remote colleagues.<br><br>

OmegaDesk provides capabilities not commercially available. It enables multiple viewers to simultaneously see stereoscopic 3D as well as 2D information from different viewpoints. It is high-resolution and multi-touch-enabled so more than one person can interact with the system at a time.<br><br>

OmegaDesk development consists of several phases. All three phases are occurring concurrently to enable successive improvements.<br><br>

<strong>Phase 1:</strong> EVL prototyped the design of a LCD-based multi-touch table, called <a href='http://www.evl.uic.edu/core.php?mod=4&amp;type=1&amp;indi=392'>TacTile</a>.<br><br>

<strong>Phase 2:</strong> While EVL&rsquo;s initial goal was to construct a stereoscopic overlay for the table, EVL&rsquo;s most recent prototype, called OmegaDesk, has two display screens - one is a multi-touch display and the other an active stereoscopic display, which are synchronized to work together. Several prototype applications now run on OmegaDesk. New camera-based input modalities (e.g., wand and gesture) are being developed.<br><br>

<strong>Phase 3:</strong> EVL is developing an API library and device drivers, called OmegaLib, as well as confidence testing the system. OmegaLib will work with TacTile as well as OmegaDesk.<br><br>

Two versions of the OmegaDesk are currently being built: stereoscopic and an autostereoscopic. The stereo version is based on a commercial product by JVC. The autostereo version is based on <a href='http://www.evl.uic.edu/core.php?mod=4&amp;type=1&amp;indi=315'>Dynallax</a>. Dynallax will enable multiple viewers to see 3D without having to wear special 3D glasses. It will enable an LCD display to support viewing in several simultaneous modes, with the viewing mode selectable on a per-pixel basis. One or two users can be head-tracked with Dynallax so that 3D information is displayed from their points of view, or multiple users can share a 3D perspective, enabling group collaboration with 3D data. No commercial product exits that can do all these modes.<br><br>

This latter mode, panoramic autostereo (or panoramagram), might be used for OmegaDesk. Multiple users could view this type of display, even upside-down, with limited &ldquo;look-around&rdquo; capability. Viewers could then stand along the two long sides of OmegaDesk and still see correct stereo views.<br><br>

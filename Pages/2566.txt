<h3>Visual Data Science: Advanced Cyberinfrastructure Systems (poster session)</h3>
<div id='entry_image_large'><img style='left:0;' src='/resizedImages/1280x720-compaas_dlv_visualdatascience_tripod.png' alt='' title='Visual Data Science: Advanced Cyberinfrastructure Systems (poster session)' width='640' height='251'></div>
<div class='entry_content'>
	<strong>Authors:</strong> <a href='/entry.php?id=178'>Brown, M.</a>, <a href='/entry.php?id=192'>Renambot, L.</a>, <a href='/entry.php?id=10'>Johnson, A.</a>, <a href='/entry.php?id=49'>Long, L.</a><br><br>
	<strong>Publication:</strong> <em>UIC Foundations of Data Science Institute (an NSF Harnessing the Data Revolution (HDR) TRIPODS grant), Open House</em>, Discovery Partners Institute, Chicago, IL<br><br>
<!-- Begin Entry Description -->
<strong>COMPaaS DLV</strong>, COMposable Platform as a Service Instrument for Deep Learning & Visualization, was purchased from Liqid, Inc., with NSF funds.  Its 4th-generation computer architecture, composable infrastructure, treats computer components - computer processing units (CPUs), graphical processing units (GPUs), networking, memory and storage - as pools of resources.  It is:<br><br>
<ul><li><strong>Highly flexible.</strong>  Each application defines the resources it needs, and the operating system composes, or combines, resources on the fly.</li><br>
<li><strong>Scalable. </strong>  As more infrastructure is added, itâ€™s auto-integrated with the existing infrastructure and becomes part of the pool of capacity.</li><br>
<li><strong>High-throughput.</strong>  System components are interconnected with a high-speed internal fabric, enabling Big Data to quickly move among CPU, GPU, memory and storage at optimum speeds with little to no bottlenecks.</ul></li><br>
Funded in part by NSF award #CNS-1828265.<br><br>

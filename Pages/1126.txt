<h3>Exploring BioPax and the Pathway Commons API Using Linked Views</h3>
<div id='entry_image_large'><img style='left:51px;' src='/resizedImages/1280x720-linked-view-screenshot.png' alt='' title='Exploring BioPax and the Pathway Commons API Using Linked Views' width='537' height='360'></div>
<span class='smallText'>Screenshot of linked-view visualization approach. - P. Murray, EVL</span><br><br>
<div class='entry_content'>
	<strong>Participants:</strong> <a href='/entry.php?id=303'>Paul Murray</a><br><br>
	Wednesday, June 10, 2015, 1PM<br>EVL Cyber-Commons, Room 2068 ERF<br><br>
<!-- Begin Entry Description -->
PhD candidate Paul Murray presents his research in biological data visualization techniques.<br><br>
The Pathway Commons web service brings together several large biological databases, with tens of thousands of biological entities and interactions that can be traversed and queried through the Pathway Commons API.  Query results are returned using the BioPax format (which is an RDF/XML OWL file), and each BioPax file contains a number of different relationship types.  Rather than try to encompass a wide variety of node types and link types into a single visualization, a linked-view visualization approach is employed that presents each type of relationship in its own view, with linked interactions between views.  This system is being integrated with the Pathway Commons API to enable effective visual queries of complex biological data.<br><br>
